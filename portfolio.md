## ✅ SEO & ATS Checklist (Fresher Edition)

### For Search Engines
- [x] Meta title includes "data science" + "fresher" + your name
- [x] Meta description mentions entry-level / looking for opportunities
- [x] Open Graph and Twitter Card tags
- [x] Canonical URL
- [x] Semantic HTML (H1 for name, H2 for sections)
- [x] Alt text on your photo
- [x] Sitemap.xml
- [x] Mobile responsive

### For ATS / HR Systems & AI Scrapers
- [x] Full name in H1 and page title
- [x] Target role clearly stated ("Aspiring Data Scientist", "Data Science Graduate")
- [x] Education: degree name, institution, year, CGPA — all in plain text
- [x] Skills listed individually as keywords (Python, SQL, Pandas, Scikit-learn, etc.)
- [x] Projects with titles, descriptions, and tech stacks — all crawlable text
- [x] Certifications with platform names and completion year
- [x] Email and LinkedIn in plain text (not just icons)
- [x] Location: city and state
- [x] "Open to opportunities" / "Actively looking" status stated
- [x] GitHub link (with repos as proof of work)
- [x] No fake job titles or inflated experience — ATS tools cross-check

### What freshers should NOT do on their portfolio
- ❌ Claim "3+ Years Experience" when you have none
- ❌ List every library you've ever heard of as "Expert"
- ❌ Include projects copied from tutorials without modification
- ❌ List "Production ML" experience without having any
- ✅ Instead: be specific, be honest, link real GitHub repos, and show what you learned
